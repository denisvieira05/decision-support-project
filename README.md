#DECISION SUPPORT APP .

Methods to assist people in making complex decisions .

Install Dependencies :
```xml
$ npm install
$ bower install
```

Run Development Enviroment :
```
$ npm install gulp -g
$ gulp
```
Running APP in localhost:8080 .
